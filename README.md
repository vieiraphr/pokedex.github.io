📱 Pokédex

Uma Pokédex interativa desenvolvida utilizando HTML, CSS e JavaScript, consumindo dados da PokéAPI para exibir informações de diferentes Pokémon.

O projeto permite navegar entre os Pokémon, visualizar nome, número e imagem de forma dinâmica.

🚀 Funcionalidades
🔍 Buscar Pokémon pelo nome ou número
⬅️ ➡️ Navegar entre Pokémon anterior e próximo
🖼️ Exibição automática da imagem do Pokémon
📱 Interface inspirada em uma Pokédex clássica
🌐 Consumo de API com JavaScript Fetch

🛠️ Tecnologias utilizadas
HTML5
CSS3
JavaScript
PokéAPI
